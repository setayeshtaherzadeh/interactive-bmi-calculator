# BMI Calculator

A simple interactive BMI calculator built using vanilla JavaScript, HTML, and CSS.

Users can input height and weight to calculate their BMI instantly with real-time updates in the UI.

---

## Features

- Real-time BMI calculation
- Simple and responsive UI
- Slider-based input controls
- Instant feedback based on BMI range

---

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

---

## Purpose

This project was built to practice basic DOM manipulation, user input handling, and UI updates without using any frameworks.
